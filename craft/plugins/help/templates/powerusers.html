##Searching Search within the CMS is a little basic at the moment, but there are several ways to make it easier to find things … ###Fuzzy searching By default, the search box within the entries section performs an ‘exact match’ search. For example a search of ‘cat’ would not return any results for ‘catfish’ or ‘cat-flap’. However it is possible to do a ‘fuzzy search’ by adding an asterisk before and after your keyword. e.g. \*cat* would return ‘catfish’ and ‘cat-flap’ ###Page searching For pages with lots of content, like a long list of entries for example, make use of your browser’s page search function. (Cmd + F on Mac, Ctrl + F on Windows) ###Frontend searching The frontend of the site provides a more powerful search tool. If you’re struggling to find the content you’re looking for, try using the main site search. If you’re logged in as an administrator we’ve added some quick links through to the Control Panel edit page for many of the articles. ##Live Preview When posting entries, do make use of ‘Live Preview’ button (top right hand side of the edit page). This lets you see any changes to the page as you make them, without the need to save and refresh the front of the site in a separate window. But always remember to close Live Preview and press ‘Save’ when you’re finished, or your changes will be forgotten! ##Related Entries You can edit related entries from within a parent entry. Just double-click the related child entry from within the list, and a small edit window will pop-up allowing you to make changes before saving. This is really useful for editing instance status from within an event. {% endset %}
